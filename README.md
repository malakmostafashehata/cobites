## 📌 Project Description

**CO-BITES** is an Object-Oriented food donation platform designed to connect volunteers, charities, and delivery personnel to reduce food waste and support communities in need.

The system provides a digital environment where food donations can be:

* Registered
* Managed
* Collected
* Delivered
* Distributed efficiently

The application demonstrates advanced **Object-Oriented Programming (OOP)** principles while solving a real-world social problem.

## ❗ Problem Statement

Food waste is a significant global issue while many communities struggle to access fresh and safe food.

Traditional donation systems suffer from:

* Lack of coordination
* Manual tracking
* Inefficient delivery processes
* No centralized management platform

---

## ✅ Proposed Solution

CO-BITES introduces a centralized food donation management system where:

* Volunteers register food donations
* Charities request available food
* Delivery personnel handle transportation
* Admin monitors system activities and users

This improves transparency, efficiency, and accessibility.

---

## 👥 System Users

### 1️⃣ Admin

* Manage users and system data
* View donations, orders, and complaints
* Monitor overall operations

### 2️⃣ Charity

* Request food donations
* Track donation history
* Submit complaints

### 3️⃣ Volunteer

Types:

* Person Volunteer
* Restaurant Volunteer
* Hotel Volunteer

Responsibilities:

* Add donations
* Manage donation records
* Update personal profile

### 4️⃣ Delivery Person

* View assigned orders
* Track delivery status

### 5️⃣ User (Abstract Class)

Base class shared by all users containing:

* Username
* Password
* Name
* Role
* Phone
